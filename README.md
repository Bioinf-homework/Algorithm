### 三大块的解题过程

- 分治
  - 描述算法的过程
    - 怎么分
    - 怎么合
  - 伪代码
    - 划分，求解，合并
  - 算法分析
    - 分析——划分，求解，合并的过程


- 动规
  - 先随便扯一下
  - 然后证明优化子结构
  - 然后给出伪代码
  - 最后分析


- 贪心
  - 先随便扯一下
  - 然后证明贪心选择性
  - 再证明优化子结构
  - 伪代码
  - 算法分析
